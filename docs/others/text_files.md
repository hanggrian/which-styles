## Insert final newline

<a
  class="label label-posix"
  href="https://pubs.opengroup.org/onlinepubs/9699919799/basedefs/V1_chap03.html#tag_03_206">
  POSIX: Line
</a>

**Always** end a file with a newline character, a common practice in Unix
systems.

Lorem ipsum.

{% endhighlight %}
